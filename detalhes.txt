dados.db tem a tabela 'dadoscriminais'
com as seguintes colunas:
colunas = {
    'id': 'INTEGER PRIMARY KEY AUTOINCREMENT',
    'NUM_BO': 'TEXT',
    'DATA_REGISTRO': 'TEXT',
    'DATA_OCORRENCIA_BO': 'TEXT',
    'HORA_OCORRENCIA_BO': 'TEXT',
    'NATUREZA_APURADA': 'TEXT',
    'LATITUDE': 'REAL',
    'LONGITUDE': 'REAL',
    'MUNICIPIO': 'TEXT',
    'BAIRRO': 'TEXT'
}




exemplo de uso para inserir ou atualizar a tabela dadoscriminais:
inserir_dados_sqlite(filtrar_colunas_excel('SPDadosCriminais_2025.xlsx', ['NUM_BO', 'DATA_REGISTRO', 'DATA_OCORRENCIA_BO', 'HORA_OCORRENCIA_BO', 'LATITUDE', 'LONGITUDE', 'NATUREZA_APURADA', 'NOME_MUNICIPIO_CIRCUNSCRIÇÃO'], -1), 'dados.db', 'dadoscriminais', 'NUM_BO')

não é necessário se preocupar em relação a dados repetidos, a função já trata isso.
